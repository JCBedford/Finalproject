About FrogPath

FrogPath is a data-driven degree planning tool built for TCU students as part of a Digital Culture and Data Analytics capstone project.

The tool combines two publicly available datasets:

  • TCU Course Catalog – course availability, prerequisites, credits, and scheduling information.
  • Rate My Professor – instructor ratings, difficulty scores, and student feedback tags filtered to TCU faculty.

Data is analyzed with Python (pandas, matplotlib) and exported as JSON files consumed by this static front-end. The site is hosted on GitHub Pages.
