About FrogPath

FrogPath is now focused on Criminal Justice classes and professors pulled from the CSV you uploaded.

The tool combines section-level class data with professor ratings and difficulty scores.

  • Class sections – course code, title, meeting time, status, enrollment cap, and seat count.
  • Professor ratings – overall rating, difficulty, and would-take-again scores where available.

Data is analyzed with Python and exported as JSON files consumed by this static front-end. The site is hosted on GitHub Pages.
